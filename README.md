## Installation 
1. Clone the repo
```
git clone https://github.com/alejleon/project_catwalk.git
```
2. Install dependencies
```
npm install
```



## Usage
To spin up the website locally, first run: 
 ```
 npm start
 ```
 followed by:
 ```
 npm run react-dev
 ```
 Go to your nearest local 3000 port to test drive the website.
